# 跑腿小程序功能总结

## 项目概述
基于 uni-app x 开发的跑腿小程序，支持用户发布订单和跑腿员接单的双端应用。

## 页面功能清单

### 用户端页面

#### 1. 首页 (pages/home/index)
**功能**：
- 顶部搜索栏（搜索服务类型、描述）
- Banner 轮播图展示
- 平台公告列表
- 服务类型网格展示（取件、送件、代购、代办等）
- 查看更多服务按钮

**样式特点**：
- 现代化卡片式设计
- 蓝色渐变背景
- 图标 + 文字组合

#### 2. 发布订单 (pages/order/create)
**功能**：
- 取件地址表单（联系人、电话、省市区选择、详细地址、地图选点）
- 送达地址选择（从地址列表选择或新建）
- 订单类型选择
- 物品描述和图片上传（最多9张）
- 费用计算（配送费、平台服务费）
- 订单提交和支付

**样式特点**：
- 表单卡片式布局
- 清晰的步骤指引
- 地图选点功能

#### 3. 订单列表 (pages/order/list)
**功能**：
- 订单状态筛选（全部、待接单、进行中、已完成、已取消）
- 订单卡片展示（状态标签、类型、地址、金额、时间）
- 下拉刷新、上拉加载更多
- 点击查看订单详情

**样式特点**：
- 列表卡片式设计
- 状态标签颜色区分
- 空状态提示

#### 4. 订单详情 (pages/order/detail)
**功能**：
- 订单状态展示和状态说明
- 收货码显示（订单创建者可见）
- 取件/送达地址详细信息
- 物品信息和图片预览
- 跑腿员信息（已接单时显示）
- 订单信息（编号、类型、时间、费用明细）
- 操作按钮：
  - 立即支付（待支付状态）
  - 取消订单（待支付/待接单状态）
  - **联系跑腿员**（服务中状态，仅显示电话联系方式）
  - 确认收货（服务中状态）
  - 评价订单（已完成状态）

**样式特点**：
- 分区块信息展示
- 清晰的信息层次
- 操作按钮明确

#### 5. 订单评价 (pages/order/evaluate)
**功能**：
- 星级评分（1-5星）
- 评价内容输入
- 提交评价

#### 6. 地址管理 (pages/address/address-list)
**功能**：
- 地址列表展示
- 新增地址按钮
- 编辑地址
- 删除地址
- 设置默认地址

#### 7. 编辑地址 (pages/address/address-edit)
**功能**：
- 联系人姓名
- 联系电话
- 省市区选择器
- 详细地址输入
- 地图选点功能
- 设置默认地址开关

#### 8. 选择位置 (pages/address/map-choose)
**功能**：
- 地图选点
- 地址搜索
- 确认位置

#### 9. 个人中心 (pages/mine/mine)
**功能**：
- 用户信息卡片（头像、昵称、手机号、角色标签）
- 功能菜单：
  - 地址管理
  - 我的订单
  - 通知中心
  - **聊天消息**（按钮，显示"聊天功能暂未开放"提示）
  - 修改密码
- 退出登录按钮

**样式特点**：
- 头部渐变背景
- 卡片式菜单设计
- 图标 + 文字组合

#### 10. 编辑资料 (pages/mine/edit-profile)
**功能**：
- 修改昵称
- 修改头像（选择图片）
- 保存资料

#### 11. 修改密码 (pages/mine/change-password)
**功能**：
- 旧密码输入
- 新密码输入
- 确认新密码
- 提交修改

#### 12. 消息列表 (pages/message/list)
**功能**：
- 系统通知列表
- 消息详情查看

#### 13. 登录 (pages/login/login)
**功能**：
- 密码登录（手机号 + 密码）
- 微信登录（一键登录）
- 注册入口
- 忘记密码入口

**样式特点**：
- 渐变背景（蓝色系）
- 现代化表单设计
- 标签页切换

#### 14. 注册 (pages/register/register)
**功能**：
- 手机号注册
- 验证码验证
- 密码设置
- 用户协议同意

#### 15. 服务页面 (pages/service/index)
**功能**：
- 所有服务类型展示
- 服务详情查看

### 跑腿员端页面

#### 1. 跑腿员首页 (pages/runner/home)
**功能**：
- 跑腿员信息展示（头像、昵称）
- 收益统计卡片：
  - 今日收益
  - 本月收益
  - 累计收益
- 平台公告列表
- 服务统计：
  - 完成订单量
  - 跑腿信用
  - 好评率
  - 服务时长
  - 服务范围
- 快捷操作入口：
  - 待接订单
  - 我的收益
  - 提现
  - 跑腿员中心

**样式特点**：
- 数据面板式设计
- 卡片式布局
- 统计信息突出

#### 2. 待接订单 (pages/runner/pending-orders)
**功能**：
- 订单状态筛选（待接单、进行中、已完成）
- 时间排序切换（从近到远/从远到近）
- 已完成订单的日期范围筛选
- 已完成订单的多种排序方式（时间、金额、距离）
- 订单列表展示（状态、类型、地址、金额、距离、时间）
- 接单操作
- 订单详情查看
- 下拉刷新、上拉加载

**样式特点**：
- 标签页切换
- 列表卡片展示
- 筛选和排序功能

#### 3. 跑腿员中心 (pages/runner/profile)
**功能**：
- 跑腿员信息展示
- 功能菜单：
  - 待接订单
  - 我的收益
  - 提现
  - 认证状态
  - **聊天消息**（按钮，显示"聊天功能暂未开放"提示）
- 退出登录

#### 4. 跑腿员认证 (pages/runner/auth-apply)
**功能**：
- 认证信息填写
- 身份证上传
- 提交认证申请

#### 5. 认证状态 (pages/runner/auth-status)
**功能**：
- 认证状态查看（待审核、已通过、已拒绝）
- 认证信息展示

#### 6. 我的收益 (pages/runner/earnings)
**功能**：
- 收益统计（今日、本月、累计）
- 收益明细列表入口
- 提现入口

#### 7. 收益明细 (pages/runner/earnings-details)
**功能**：
- 收益记录列表
- 筛选和排序
- 收益详情查看

#### 8. 提现 (pages/runner/withdrawal)
**功能**：
- 可提现金额展示
- 提现金额输入
- 提现方式选择（银行卡、微信、支付宝）
- 提交提现申请

#### 9. 提现记录 (pages/runner/withdrawal-records)
**功能**：
- 提现记录列表
- 提现状态查看（待审核、已到账、已拒绝）

### 公共页面

#### 1. 隐私政策 (pages/common/privacy)
**功能**：隐私政策内容展示

#### 2. WebView (pages/common/webview)
**功能**：通用网页展示

## 设计风格

### 颜色方案
- **主色调**：蓝色渐变 (#2563eb → #3b82f6 → #60a5fa)
- **辅助色**：
  - 绿色 (#10b981, #34d399) - 用于成功状态、收益等
  - 橙色 (#f97316, #fb923c) - 用于订单、警告等
- **背景**：白色卡片 + 渐变背景

### UI 特点
- 卡片式设计
- 圆角按钮（50rpx）
- 渐变背景和按钮
- 图标 + 文字组合
- 清晰的信息层次
- 状态标签颜色区分

### 交互特点
- 下拉刷新
- 上拉加载更多
- 状态标签区分
- 操作按钮明确
- 空状态提示
- 加载状态提示

## 技术栈
- **框架**：uni-app x
- **语言**：Vue 3 + TypeScript/UTS
- **平台**：微信小程序
- **地图**：高德地图
- **支付**：微信支付、支付宝
